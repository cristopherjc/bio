# BIO Web Page
Página web Biográfica correspondiente a la Actividad: Diseño de Página Web Personal con Formulario de Contacto en PHP
de la Matería Desarrollo Web del 5to ciclo de Ingeniería en Tecnologías de la información

Universidad Técnica Particular de Loja

El proyecto fue desarrollado en PHP, tiene un script de js para que funcione como un SPA (Single Page Application) porque al agregar una radio me di cuenta que cada vez que se cambiaba de página se cortaba la música.
También esto ayuda al rendimiento de la página.

El link de la aplicación es: https://schoolboyish-unenigmatic-earl.ngrok-free.dev
Al entrar por primera vez saldrá un aviso de Ngrok, esto es por el uso de la capa gratuita.

La aplicación es redirigida por tunel desde mi raspberry hasta ngrok y asi es servida en internet.
